# AI/LLM Briefing — 17 July 2026

## Telegram — Run 07:00

Moonshot AI launched Kimi K3, a 2.8-trillion-parameter model that runs only 16 of its 896 expert sub-networks per query — a mixture-of-experts design, so most of it stays idle each call. It has a 1-million-token context window (the amount of text it reads at once). Moonshot reports 88.3 on Terminal-Bench 2.1, a command-line coding benchmark. Open weights — free to download and run — arrive by 27 July; independent results pending.

"the world's first open 3T-class model"

Output runs $15 per million tokens.
